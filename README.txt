This VR target shooting game, built in Unity with WebXR support, lets players use a bow and arrow to hit randomly spawned targets. 
It features procedural target generation within defined boundaries, collision-based scoring, showcasing essential VR and procedural design techniques.

This project is free to use for learning and non-commercial purposes. For other uses, please don't contact the author.
